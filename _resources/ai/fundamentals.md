## Artifical Intelligience vs Machine Learning 


### Artificial Intelligience


**Aritificial Intelligence** is primarily used for answering questions, and 
performing tasks thaty may otherwise take a long time to perform. AI is 
also commonly used as as tool. We will look more into how to build an AI 
system known as the **Retrieval-Augmented Generation** (RAG) later.


### Machine Learning


**Machine Learning** is a type of ai that we can't use as much as a tool or
resource for answering questions. Despite this, machine learning is still very useful in 
taking in data to make predictions about the future and unknown values with. 
What makes machine learning systems special is that they are able to learn while 
making predictions to become more accurate over time. We will look more into this later.
